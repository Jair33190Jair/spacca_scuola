SpaccaScuola

Trasforma registrazioni vocali di lezioni in materiale di studio strutturato. Una guida passo-passo per la sessione di setup.

⏱ 4 ore di sessione 💰 costo quasi zero
00

La Pipeline

🎙 Registrazione
Voice Memo / telefono
→
📝 Trascrizione
Whisper (gratis)
→
🤖 Elaborazione
Claude Code
→
📚 Output
4 formati di studio

📋 I 4 output generati

  • Riassunto breve — versione condensata della lezione (1-2 pagine max)
  • Bullet points — concetti chiave in punti elenco gerarchici
  • Tabella concetti — contenuto organizzato in tabelle (concetto | definizione | collegamento)
  • Mappa mentale — struttura temi + relazioni pronta per essere disegnata a mano
Filosofia: L'obiettivo NON è sostituire lo studio, ma dare una struttura chiara da cui partire. Ridisegnare le mappe mentali a mano è il vero atto di apprendimento.

Pre-requisiti (prima della sessione)

✅ Checklist preparazione

  • macOS aggiornato (Sonoma o più recente consigliato)
  • Installare Homebrew: aprire Terminal e incollare il comando sotto
  • Avere una registrazione audio di prova (anche 5 minuti bastano)
  • Creare un account su anthropic.com (serve per l'API key di Claude)
  • Opzionale: creare un account GitHub (per backup e collaborazione futura)
# Installa Homebrew (gestore pacchetti per macOS) /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
01

Setup dell'ambiente

⏱ ~60 min

🔧 Step 1: Installa gli strumenti base

Apri Terminal (Cmd+Spazio → scrivi "Terminal") e copia questi comandi uno alla volta:

# 1. Installa Node.js (serve per Claude Code) brew install node # 2. Installa Whisper (trascrizione vocale gratuita) brew install openai-whisper # oppure la versione più veloce: pip3 install whisper-ctranslate2 --break-system-packages # 3. Installa ffmpeg (necessario per elaborare audio) brew install ffmpeg # 4. Installa Claude Code npm install -g @anthropic-ai/claude-code

📁 Step 2: Crea la struttura del progetto

# Crea la cartella del progetto mkdir -p ~/SpaccaScuola/{audio,trascrizioni,output,archivio} cd ~/SpaccaScuola
📂 SpaccaScuola/
├── 📂 audio/ ← registrazioni originali
├── 📂 trascrizioni/ ← testo trascritto da Whisper
├── 📂 output/ ← materiale di studio generato
├── 📂 archivio/ ← lezioni precedenti (contesto)
├── CLAUDE.md ← istruzioni personalizzate per Claude
├── trascrivi.sh ← script trascrizione
└── elabora.sh ← script elaborazione

🔑 Step 3: Configura l'API key

Vai su console.anthropic.com → API Keys → Create Key, poi:

# Salva la chiave nel tuo profilo (da fare una volta sola) echo 'export ANTHROPIC_API_KEY="sk-ant-la-tua-chiave-qui"' >> ~/.zshrc source ~/.zshrc
Attenzione: La chiave API è come una password. Non condividerla e non metterla in file pubblici.
02

Configura la pipeline

⏱ ~60 min

📄 Step 4: Crea il file CLAUDE.md

Questo file è il cervello del sistema — dice a Claude come elaborare le tue lezioni. Copialo nella cartella del progetto.

Importante: Questo file è già incluso nel pacchetto starter che puoi scaricare. Lo personalizzeremo insieme durante la sessione.

Il file CLAUDE.md contiene:

  • Il contesto della materia (es. Scienze Politiche, Sociologia, ecc.)
  • Le istruzioni per ogni tipo di output
  • I comandi rapidi (/riassumi, /bullet, /tabella, /mappa)
  • Il collegamento con lezioni precedenti

🎙 Step 5: Lo script di trascrizione

Questo script automatizza la trascrizione. Basta trascinare il file audio nel Terminal.

# Uso quotidiano — un solo comando: ./trascrivi.sh audio/lezione-2024-03-15.m4a # Output: trascrizioni/lezione-2024-03-15.txt

🤖 Step 6: Lo script di elaborazione con Claude

Dopo la trascrizione, un secondo comando genera tutto il materiale di studio:

# Elabora e genera tutti i 4 formati: ./elabora.sh trascrizioni/lezione-2024-03-15.txt # Output generati in output/lezione-2024-03-15/: # riassunto.md # bullet-points.md # tabella-concetti.md # mappa-mentale.md
Flusso quotidiano completo: Registra → ./trascrivi.sh audio/file.m4a → ./elabora.sh trascrizioni/file.txt → studia! Due comandi e basta.
03

Test con lezione reale

⏱ ~60 min

🧪 Step 7: Esegui la pipeline completa

  • Usa una registrazione vera di una sua lezione
  • Fai girare la pipeline end-to-end
  • Esamina ogni output — è utile? Manca qualcosa?
  • Modifica il CLAUDE.md per calibrare la qualità

🎯 Step 8: Calibrazione dei prompt

Questa è la fase più importante. Per le scienze sociali, gli aggiustamenti tipici sono:

  • Aggiungere istruzioni per catturare nomi di autori e teorie
  • Differenziare tra fatti, opinioni del professore, e dibattito aperto
  • Collegare i nuovi concetti con le lezioni precedenti
  • Adattare il livello di dettaglio delle tabelle

🔄 Step 9: Impara ad usare Claude Code interattivamente

Oltre agli script, Claude Code si può usare in modo conversazionale:

# Apri Claude Code nella cartella del progetto cd ~/SpaccaScuola claude # Ora sei in modalità interattiva. Prova: > Leggi la trascrizione in trascrizioni/lezione-01.txt e fammi un riassunto > Confronta questa lezione con la precedente in archivio/ > Quali temi si ripetono tra le ultime 3 lezioni?
04

Manutenzione e autonomia

⏱ ~60 min

📖 Step 10: Documenta e spiega il sistema

  • Il flusso completo: registra → trascrivi → elabora → studia
  • Come modificare CLAUDE.md per migliorare i risultati
  • Come aggiungere contesto dalle lezioni passate
  • Troubleshooting: cosa fare se qualcosa non funziona

💾 Step 11: Backup con Git (versione minima)

Solo 3 comandi da ricordare, niente di più:

# Primo setup (da fare una volta) cd ~/SpaccaScuola git init git add . git commit -m "setup iniziale" # Ogni volta che vuoi salvare il progresso: git add . git commit -m "aggiunta lezione del 15 marzo" # Per backup su GitHub (opzionale): git push
Per ora basta questo. Branching, merge e pull request possono aspettare. L'obiettivo è non perdere il lavoro.

🔮 Step 12: Piano per il futuro

  • Settimana 1-2: Usa il sistema per 4-5 lezioni. Annota cosa va migliorato.
  • Settimana 3: Sessione di 1h con te per calibrare CLAUDE.md.
  • Mese 2: Aggiungere il confronto automatico tra lezioni per preparazione esami.
  • Futuro: Automazione completa con Automator/shortcut macOS.

GitHub GRATIS Repo privati gratuiti

🆓 Alternativa a costo zero

Se anche €1-3/mese è troppo, può usare claude.ai (piano gratuito) e incollare manualmente la trascrizione nel chat. Meno automatizzato, ma funziona. Basta salvare i prompt in un file di testo da copiare-incollare.

Framework replicabile

♻️ Per replicare con altri studenti

Il setup è pensato per essere clonabile. Per un nuovo studente basta:

  • Copiare la cartella SpaccaScuola
  • Modificare CLAUDE.md con la sua materia e preferenze
  • Configurare la sua API key
  • Fatto — stesso flusso, contesto diverso

📈 Evoluzione possibile

  • Fase 2: macOS Shortcut / Automator — drag & drop del file audio genera tutto
  • Fase 3: Web app locale con interfaccia semplice
  • Fase 4: Quiz generator basato sulle lezioni precedenti
  • Fase 5: Confronto multi-lezione per preparazione esami
